Record Type list to modify

Go to the Record Type of the sObject agf__ADM_Work__c.

Setup > Object Manager > agf__ADM_Work__c

On the left side of your screen go to 'Record Types', underneath Details.

The folowing types will be present :

  • Bug
  • Investigation
  • Template
  • ToDo
  • User Story

Only Bug and User Story are necessary. Edit all other record type, and uncheck the 'Active' checkbox. You'll have to do it for the folowing list :

  • Investigation
  • Template
  • ToDo

Devide the 'Status' List in three categories : available by all, available by Bugs only, available by User Stories only

available by all

If you follow the steps I will describe this list will be set by default.

Set the default value of any picklist

Now that your list of active Record Type is down to Bug and User Story, click on the first one. In 'Picklists Available for Editing', look for 'Priority' and 'Status'. Edit both of them. In section 'Picklist Values', at the buttom, above save buttun, you'll find a picklist named 'Default'. Set it to 'P1' or 'New'.
