List of new labels to create

You could always imagin updating this project by creating a sObject Duration__c and a sObjcet SlA__c. But this version does not support such design. Field Label : Flag

API Name : Flag__c

Description : When a record time of response or corrective maintenance exceeds the service level agreements this field will be 'YES', and it's color will change from green to red.

Help : This picklist is automaticaly set by the trigger when the timing agreed on the service level agreements is exceeded.

Type : Picklist

Enter values, with each value separated by a new line :

  • no
  • YES

  • Display values alphabetically, not in the order entered
  • Use first value as default value
  • Restrict picklist to the values defined in the value set

Default Value : Unchecked

This field can be seen by all. And Read only is applied in Layout.

Layouts : Added to All layouts

After you have saved the field, go to the detail page of this field. In the section Values make sure 'No' is the default value.

  • Clic on the button Edit to 'No', to assign dynamically the color green to this value in 'Chart Color'.
  • Clic on the button Edit to 'YES', to assign dynamically the color red to this value in 'Chart Color'.
